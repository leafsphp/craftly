Craftly is the modern, open platform for creating and publishing websites, blogs, and pages. Simple for creators, powerful for developers, and crafted for the modern web.

## Setting up

Install the Craftly module in your Leaf app

```bash
leaf install craftly
```

That's it! You will now be able manage your site through the craftly dashboard.
